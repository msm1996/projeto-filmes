<h1>🎬 Catálogo de Filmes</h1>
<h2>Aplicação desenvolvida com React.js que consome a API do TMDB para exibir filmes e séries em um catálogo interativo.</h2>
<br>

<h1>🚀 Sobre o projeto<h1/>
<h2>Este projeto foi criado com o objetivo de praticar conceitos modernos de desenvolvimento front-end, simulando uma aplicação real de mercado.</h2>
<h2>O usuário pode navegar por um catálogo de filmes e, ao clicar em um item, visualizar informações detalhadas como imagem, sinopse e trailer.</h2>
<br>
<h1>🛠️ Tecnologias utilizadas:</h1>
<h2>⚛️ React.js
  <br>
🎨 Styled Components (ou CSS Modules — ajuste aqui)
   <br>
🌐 API do TMDB
   <br>
📦 Axios
   <br>
🔀 Componentização</h2>
  <br>
<h1>💡 Funcionalidades:</h1>
<h2>📃 Listagem de filmes e séries
   <br>
🔍 Exibição de detalhes ao clicar no filme
   <br>
🎥 Trailer integrado
   <br>
🖼️ Banner com imagem e sinopse
   <br>
♻️ Reaproveitamento de componentes</h2>
